# Contributing to ResumeAI

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Follow the setup steps in `README.md` for both `client` and `server`
4. Create a new branch for your change: `git checkout -b feature/your-feature-name`

## Making Changes

- Keep commits small and focused on one change
- Write clear commit messages (e.g. `fix: resolve upload validation bug`)
- Test your changes locally before submitting

## Submitting

1. Push your branch to your fork
2. Open a Pull Request against `main`
3. Describe what you changed and why

## Code Style

- Use consistent formatting with the rest of the codebase
- Keep functions small and single-purpose
- Comment non-obvious logic

Questions or issues? Open a GitHub Issue.
